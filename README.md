# UpTask - Administración de Proyectos

## Descripción

UpTask es una aplicación que permite a los usuarios crear sus proyectos, agregando colaboradores y tareas.

## Tecnologías utilizadas

### FrontEnd

- React
- React Router Dom
- Redux
- Formik
- Zustand

### Backend

- Node.js
- Express
- bycrypt
- jsonwebtoken
- Jest
- SuperTest
- MongoBD
- Mongoose

## Lenguaje

- TypeScript
